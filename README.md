# Shopy Products Sales Analysis – Power BI

## 📌 Project Overview

This project presents an interactive Sales Analysis Dashboard built using Microsoft Power BI. It analyzes sales performance of Shopy Products and provides insights into revenue trends, product performance, and customer engagement.

## 📊 Dashboard Features

### KPI Metrics

* YTD Sales – $2.18M
* QTD Sales – $811.1K
* YTD Products Sold – 27.75K
* YTD Reviews – 19.42M

### Sales Trend Analysis

* Monthly Sales Trend
* Weekly Sales Distribution

### Category Performance

* Sales by Product Category
* Percentage Contribution
* Revenue Comparison

### Top Performers

* Top 5 Products by Sales
* Top 5 Categories by Revenue

### Interactive Filters

* Quarter Filter
* Category Slicers
* Dynamic Dashboard Controls

## 🛠️ Tools Used

* Microsoft Power BI
* Data Modeling
* DAX Calculations
* Interactive Visualizations

## 📈 Key Insights

* Strong sales growth in recent periods
* Men’s fashion categories generate highest revenue
* Few products contribute major share of total sales
* Weekly sales patterns reveal peak periods

## 🎯 Business Objective

To monitor sales performance, identify top-selling products, analyze category contribution, and support data-driven decision making.
